First Java Code
/* Hello World
 * CSC 230
 */

public class HelloWorld
{//start brace

	// main function, starting point of running
	// String args[], parameters
	public static void main(String args[])
	{
		// print out message to standard output
		System.out.println("Hello World!");// semiicolon
	}

}//end brace
Syntax
  • Comments
    • /* ... */, block comment
    • //, line comment
  • {}, delimit the body of a class or method
  • System.out.println, display a line of string
  • "...", a string
  • ;, end of a line
  • Java is case sensitive
