Garbage collector and finalize method
public class GC
{
	private String name;
	private static int count = 0;

	public GC(String n)
	{
		name = n;
		count++;
	}
	public String getName() {return name;}
	public void setName(String n) {name = n;}

	public static int getCount() {return count;}

	protected void finalize()
	{
		count--;
		System.out.printf("Remove object, %d\n", count);
	}

	public static void main(String args[])
	{
		GC g1 = new GC("Mike");
		GC g2 = new GC("David");

		System.out.printf("%d objects have been created\n", GC.getCount());

		g1 = null;
		g2 = null;

		System.out.println("Both have been marked as removed");

		System.gc();

		System.out.printf("Has %d objects in the memory\n", GC.getCount());
	}
}
  • Garbage collector is used to release the memory of objects without reference
  • finalize method is called by the garbage collector to perform termination housekeeping
  • Garbage collector may never execute before a program terminates

public class Static
{
	public static void main(String args[])

		double n = 10;

		System.out.println(sqrt(n));
	}
}
