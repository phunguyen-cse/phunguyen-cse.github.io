Static Method, Static Fields
//S.java
public class S
{
	public static void f()
	{
		System.out.println("f() function");
	}

	public static void f2()
	{
		System.out.println("f2() function");
	}

	public static final int N = 100;
}
			
public class STest
{
	public static void main(String args[])
	{
		S.f();
		S.f2();
		System.out.println(S.N);
	}
}
static methods and static fields are special class methods and fields that are able to be accessed by ClassName.Method or ClassName.Field, they are shared by all objects declared by the class
Shared static methods and static fields
//Student.java
public class Student
{
	public static int num = 0;//stattic field
	private int grade;//instance field
	private String name;//instance field

	// constructor
	public Student()
	{
		num++;
	}

	public Student(String name, int grade)
	{
		this.grade = grade;
		this.name = name;
		num++;
	}

	// static methods
	public static int getNum()
	{
		return num;
	}

	// accessor
	public int getGrade() {return grade;}
	public String getName() {return name;}

	// mutator
	public void setGrade(int grade)
	{
		this.grade = grade;
	}

	public void setName(String name)
	{
		this.name = name;
	}

	public static void main(String args[])
	{
		System.out.println(Student.getNum());

		Student s1 = new Student("David", 90);
		Student s2 = new Student("Mike", 80);

		System.out.printf("%d students have been created\n", Student.getNum());
	}
}
  • instance methods can access instance variables and instance methods
  • instance methods can access static fields and static methods
  • static methods can access static fields and static method
  • static methods cannot access instance variables or instance methods
